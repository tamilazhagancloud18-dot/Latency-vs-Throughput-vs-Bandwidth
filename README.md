## 📖 Overview

This infographic explains the differences between:

- ⚡ **Latency** – Time taken for a request to travel from client to server and back.
- 🚀 **Throughput** – Amount of data or requests processed per second.
- 🌐 **Bandwidth** – Maximum data transfer capacity of a network.

### Real-world Analogy

- 🚗 Latency = Travel time
- 🛣️ Bandwidth = Number of highway lanes
- 🚙 Throughput = Cars actually passing per minute

### DevOps / SRE Example

| Metric | Example |
|---------|---------|
| Latency | 50 ms |
| Throughput | 10,000 Requests/sec |
